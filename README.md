# TrustHerBro

Scaffold ban dau cho ung dung Next.js App Router gom:

- Landing page va mode entry
- First Night onboarding
- Survival Timeline
- Safety Mode
- Assistant va Emergency stubs
- API routes mock de noi logic AI/TTS sau

## Chay local

```bash
npm install
npm run dev
```

## Bien moi truong

Tao gia tri trong `.env.local`:

- `OPENAI_API_KEY`
- `VBEE_API_KEY`
- `MAPS_API_KEY`
